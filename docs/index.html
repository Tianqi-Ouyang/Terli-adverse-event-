Author

Tianqi Ouyang

Published

Overview

This sub-project of the HARMONY Terlipressin Registry analyzes adverse events (AE) associated with terlipressin treatment in patients with hepatorenal syndrome-acute kidney injury (HRS-AKI) across 15 hospitals.

Data Pipeline

Adverse Event Definitions

Variable Definition Type
any_ae Any adverse event Categorical
abdominal_ae abdpain OR diarrhea OR nausea Categorical
serious_ae respfail OR brady OR terli_7 OR terli_8 Categorical

3. Logistic Regression — Predictors of AE

  • Univariable: scr_terli_day0, respiratory_f, aclf_grade → outcome any_ae
  • Multivariable (original): aclf_grade + scr_terli_day0 + respiratory_f → outcome any_ae
  • Multivariable (04/07/2026 update): sex_male + inr_admit + akin + chest_imaging_effusion → outcome any_ae

4. Competing Risk Models — 90-Day Mortality

Univariable (Fine-Gray, failcode=1 death): All individual AE types + serious_ae

Multivariable (4 models):

Model Covariates
1 any_ae + age + race + hispanic + admit_meld_3 (updated 04/07/2026, was day0_meld)
2 any_ae + age + race + hispanic
3 serious_ae + age + race + hispanic + admit_meld_3
4 serious_ae + age + race + hispanic

5. Subgroup: ICU Admission

  • Cohort: location_terli_day0 == 1 OR 2
  • Model: Univariable logistic — any_ae → icu_admit

6. Subgroup: Transplant-Listed Patients

  • Cohort: listed_transplant == 1
  • Model: Univariable competing risk (failcode=2, transplant)
  • Covariates: any_ae, serious_ae, hrs_responders_cat_2, day0_meld, aclf_grade, meld_fda
